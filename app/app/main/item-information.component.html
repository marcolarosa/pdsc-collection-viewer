    Zotero and Mendeley Integration.

    This page has embedded metadata that is able to be read by both Zotero and Mendeley.

    Zotero users with the appropriate browser plugin ( https://www.zotero.org/download/) can save to Zotero directly whilst Mendeley users should install the one click Web Importer (https://www.mendeley.com/import/) to enable the same functionality.
